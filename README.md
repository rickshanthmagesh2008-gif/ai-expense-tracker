# 💰 AI Expense Tracker

An AI-powered Expense Tracker built using **n8n**, **OpenAI GPT-4.1 Mini**, **Google Sheets**, and **Simple Memory**. It automatically extracts expense details from natural language and records them into Google Sheets.

## 🚀 Features

- 💰 AI-Powered Expense Tracking
- 📊 Automatic Google Sheets Integration
- 🤖 Natural Language Expense Extraction
- 🧠 Conversational Memory
- ⚡ Workflow Automation with n8n
- 📅 Automatic Date Recording

## 🛠️ Tech Stack

- n8n
- OpenAI GPT-4.1 Mini
- Google Sheets
- Google OAuth2
- Simple Memory

## 📦 Installation

1. Download the `ai-expense-tracker.json` workflow.
2. Import the workflow into n8n.
3. Connect your OpenAI Chat Model.
4. Connect your Google Sheets account.
5. Create a Google Sheet with these columns:
   - Date
   - Category
   - Amount
   - Description
6. Save and activate the workflow.
